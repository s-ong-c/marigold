NumberField

Description

The NumberField components allows users to enter numbers with a keyboard. This includes plain numbers as well as percentages, decimals and sums of money. It consists of a label, an input field and a help text. The later also includes error messages.

You should really try this!
import { NumberField } from '@marigold/components';
NameTypeDefaultDescription
label (optional)ReactNodeThe label text. If you don't want to visually display a label, provide an aria-label or aria-labelledby attribute for accessibility.
description (optional)ReactNodeA helpful text
errorMessage (optional)ReactNodeAn error message
error (optional)booleanfalseIf true, the field is considered invalid and if set the errorMessage is shown instead of the description.
value (optional)numberThe value of the input field
disabled (optional)booleanfalseIf true, the input is disabled.
required (optional)booleanfalseIf true, the input is required.
readOnly (optional)booleanfalseIf true, the input is readOnly.
onChange (optional)functionA callback function that is called with the input's current value when the input value changes.
formatOptions (optional)Intl.NumberFormatOptionsApply formatting. See Intl.NumberFormatOptions for available options.
width (optional)string100%Control the width of the field.
...Yes you can use all regular attributes of input!

Examples

Simple Number Field

<NumberField label="Numbers" />

Disabled Number Field

<NumberField label="Field disabled" disabled />

Required Number Field

<NumberField label="Numbers" required />

Number Field with an Error

<NumberField label="Amount" error errorMessage="Something went wrong!" />

Min and max Values

<NumberField label="Length" defaultValue={55} minValue={0} maxValue={100} />

Step Amount

<NumberField label="Length" defaultValue={10} step={10} />

Hide Stepper

<NumberField label="Numbers" hideStepper />

Diplay a Decimals

<NumberField
  label="Decimals"
  formatOptions={{
    signDisplay: 'exceptZero',
    minimumFractionDigits: 1,
    maximumFractionDigits: 2,
  }}
  defaultValue={0}
/>

Diplay a Currency

<NumberField
  label="Amount"
  defaultValue={19.99}
  formatOptions={{
    style: 'currency',
    currency: 'USD',
  }}
/>

Diplay a Percentage

<NumberField
  label="Percentage"
  defaultValue={0.42}
  formatOptions={{
    style: 'percent',
  }}
/>

Display a Unit

<NumberField
  label="Length"
  defaultValue={150}
  minValue={0}
  formatOptions={{
    style: 'unit',
    unit: 'centimeter',
    unitDisplay: 'short',
  }}
/>
