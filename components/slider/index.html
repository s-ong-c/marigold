Slider

The Slider component allows users to quickly select a value within a range. They should be used when the upper and lower bounds to the range are invariable.

50%
import { Slider } from '@marigold/components';

Props

PropertyTypeDefaultDescription
disabled (optional)booleanfalseIf true, the slider is disabled.
value (optional)number[]The slider's value (controlled).
defaultValue (optional)number[]The slider's default value (uncontrolled).
maxValue (optional)number100The slider's maximum value.
step (optional)numberThe slider's step value.
formatOptions (optional)Intl.NumberFormatOptionsThe slider's step value.
width (optional)string100%Control the width of the field.
...Yes you can use all regular attributes of input type = range!

Examples

Width

<Slider width={180} aria-label="width">
  Example
</Slider>

Disabled

<Slider disabled>Example</Slider>

Step

<Slider step={10}>Example</Slider>

Max value

<Slider aria-label="maxValue" maxValue={50}>
  Example
</Slider>

Value (controlled)

() => {
  const [value, setValue] = React.useState([75]);
  return (
    <Slider value={value} onChange={setValue}>
      Example
    </Slider>
  );
};

Default Value (uncontrolled)

<Slider defaultValue={[25]}>Example</Slider>

Currency

<Slider formatOptions={{ style: 'currency', currency: 'EUR' }}>Price</Slider>

Percent

<Slider formatOptions={{ style: 'percent' }} step={0.01} maxValue={1}>
  Percent
</Slider>
